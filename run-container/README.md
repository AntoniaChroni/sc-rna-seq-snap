# rstudio_containers

This is an example startup sequence:
 
```
ssh hpc.stjude.org
hpcf_interactive -n 4
cd /lustre_scratch/shared_scratch/wabuala/rstudio_containers/
bash run.sh
```

When RStudio launches, please click "Session" -> "Restart R".
